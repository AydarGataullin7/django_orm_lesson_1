
# Пульт охраны банка

Это внутрениий репозиторий для сотрудников банка "Сияние". Если вы попали в это репазиторий случайно, то вы не сможете его запустить, т.к. у вас нет доступа к БД, но можете свободно использовать код верстки или посмотреть как реализованы БД.

Пульт охраны - это сайт, который можно подключить к удаленной базе данных с визитами и карточками сотрудников нашего банка. 

## Как установить

Перед запуском программы необходимо настроить переменные окружения. Создайте файл .env в корневой директории проекта и укажите следующие переменные:

1. DB_HOST - хост базы данных 
1. DB_PORT - порт базы данных
1. DB_NAME - название базы данных
1. DB_USER - пользователь базы данных
1. DB_PASSWORD - пароль пользователя базы данных
1. DB_SECRET_KEY - секретный ключ для Django

Также, убедитесь, что установлены необходимые пакеты:

````pip install django psycopg2-binary python-dotenv````

После настройки переменных окружения запустите программу:

````python main.py````

**Описание работы:**
Программа подключается к базе данных PostgreSQL используя указанные настройки, и выводит:
Общее количество пропусков в системе
Количество активных пропусков (со статусом is_active=True)

**Пример работы:**
````
PS C:\Users\aydar\Desktop\python_scripts\django_orm_lesson_1> python main.py
Количество пропусков: 100
Количество активных пропусков: 89
````
